# 📦 N_Inventory

Lightweight inventory management system based on a single web page (HTML/JavaScript), with **local persistence** and **Dark/Light Mode** functionality, adaptable to desktop and mobile devices.

---

## ⚡ Main features

- **Data persistence:** Data is automatically saved in the browser's local storage (`localStorage`).  
- **CRUD operations:** Add, remove, increase, and decrease item quantities.  
- **Global Reset:** Button to set all inventory quantities to zero for planning the next period.  
- **Quick Search:** Filter by item name or type.  
- **Visual Themes:** Switch between Dark Mode (default) and Light Mode (Sun/Moon ☀️/🌙).  
- **Sorting:** Sort the table by Item, Type, or Quantity.  
- **Device-friendly:** Interface optimized for **desktop** and **mobile**, with responsive design.

---

## 🛠️ Technologies

- **HTML5 / CSS3** – Structure and styling (Dark Mode and Light Mode, responsive design).  
- **JavaScript ES6+** – Logic, DOM manipulation and `localStorage` management.  

---

## 🚀 How to use

To start using N_Inventory:

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/fvilpaz/N_Inventory
   ```
2. **Navigate to the project folder:**

       cd N_Inventory

3. **Choose the version according to your device:**

    - Desktop: For large screens.

    - Mobile: For mobile devices.

4. **Open the html file corresponding to the version you chose in your browser.**

## 📝 Note

This project runs locally, with no server required.

Persistence depends on the browser and is not intended for shared production environments without a dedicated backend.

The interface adapts automatically if you choose the mobile version, making the experience more comfortable on small screens.

---

## 📌 Goal

To provide a simple, fast solution for stock control, accessible without servers or complex installations, on both desktop and mobile devices.
